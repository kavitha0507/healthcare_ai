# 🩺 MediSync AI: Agentic Health Advocacy

**MediSync AI** is a high-fidelity, agent-driven healthcare interface designed to bridge the gap between raw medical data and patient wellness. Built with a focus on **Agentic UX** and **Universal Design**, it provides proactive health insights and local specialist matching.

## ✨ Key Features
* **Agentic Conversational UI:** A proactive LLM-powered advocate that interprets BMI and health queries.
* **Health Memory (Activity History):** A persistent sidebar that maintains context across sessions.
* **Visual Delights:** Positive reinforcement through micro-interactions (Confetti on healthy milestones).
* **Austin Specialist Matching:** Localized health networking built directly into the agent's logic.

## 🎨 Design System
* **Glassmorphism:** Modern translucent UI with a `24px` blur factor to reduce clinical anxiety.
* **Accessibility:** Adherence to WCAG guidelines using high-contrast typography and semantic structure.
* **Color Palette:** Trust-based Blue (`#3b82f6`) and Innovation-based Purple (`#a855f7`).

## 🛠️ Architecture
- **Frontend:** React + Vite + Tailwind v4 (Deployed on Vercel)
- **Backend:** FastAPI + Python (Deployed on Render)
- **Intelligence:** Agentic reasoning for health diagnosis interpretation.
